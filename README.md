# Simple Pokédex

## 📌 Description
Simple Pokédex is a web application that allows users to explore Pokémon data using a clean and interactive interface. It fetches real-time data from the PokéAPI and provides features like search, filtering, sorting, and interactive card expansion.

---

## 🚀 Features

### 🔍 Search
- Search Pokémon by name (case-insensitive)

### 🎯 Filter
- Filter Pokémon by type (Fire, Water, Grass, Electric)

### 🔃 Sorting
- Sort Pokémon alphabetically (A → Z, Z → A)

### ⭐ Favorite System
- Mark/unmark Pokémon as favorites
- Visual feedback using star toggle

### 📊 View More (Card Expansion)
- Click a card to view:
  - Height
  - Weight
  - Stats (HP, Attack, etc.)
- Only one card expands at a time

### 🎞️ Animated GIFs
- Pokémon shows animated sprite when expanded
- Falls back to static image if GIF fails

### 🌗 Dark Mode / Light Mode
- Toggle theme for better user experience

---
