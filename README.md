# Pokédex Explorer

## Description
Pokédex Explorer is a web application that allows users to explore different Pokémon using data fetched from a public API. Users can search for Pokémon, filter them by type, and sort them based on different criteria.

## API Used
PokéAPI (https://pokeapi.co/)

## Features
- Search: Search Pokémon by name
- Filter: Filter Pokémon by type
- Sort: Sort Pokémon by name

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open index.html in a browser
